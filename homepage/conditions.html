The C language has its own syntaxe to exectute some instructions under one condition
The syntaxe for using conditions in your code is:

if (A > 2)
{
    B = A + 2 ;
}

when the statement in red is true the instructions in green are going to be executed

you can also use:

if (A > 2)
{
    B = A + 2 ;
}
else
{
    B = A - 2 ;
}

when the statement is not true, the instructions between the else brackets are the ones that are getting executed.
If you want to use the true and false parameters in your statement, you need to include the library <bool.h> But you can always use the numbers 1 and 0 instead.
